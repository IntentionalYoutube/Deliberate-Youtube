# Deliberate-Youtube
I want to help people escape the addictive algorithms of Youtube. I don't think there's anything wrong with entertainment, but I despise how it makes you lose track of time and drags you into a cycle of procrastination.

If you get a "failed to load" popup in chrome://extensions/, try to click inside the selected folder (the extension folder may be inside another folder with both having the same name "Deliberate-Youtube-main"). You know if you selected the right folder if opening it reveals "path" and "tests."

If you encounter a bug and are using the extension as a bug tester, please make a note in the forms.

Instructions: 

Install the extension. 

Choose the intent that best matches why you're opening YouTube. 

Use YouTube naturally. 

Don't try to "game" the alignment score. 

Ignore the debug panel unless instructed. 

At the end of the study, complete the Google Form.
